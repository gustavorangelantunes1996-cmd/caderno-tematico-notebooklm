Caderno Temático — Inteligência Artificial Aplicada aos Estudos com NotebookLM
📚 Sobre o Projeto
Este projeto foi desenvolvido como desafio prático da DIO com o objetivo de explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa utilizando o NotebookLM.

O foco do estudo foi compreender como a IA pode auxiliar estudantes e profissionais na organização do conhecimento, criação de resumos, revisão de conteúdos e construção de materiais de apoio para estudos.

Além disso, o projeto também demonstra práticas de:

Curadoria de fontes;
Engenharia de prompts;
Organização de informações;
Pensamento crítico;
Documentação técnica no GitHub.
🎯 Objetivos do Estudo
Os principais objetivos deste caderno temático foram:

Entender o funcionamento do NotebookLM;
Aprender como utilizar IA para melhorar os estudos;
Criar resumos automáticos com apoio da IA;
Desenvolver prompts eficientes;
Organizar conteúdos de maneira estruturada;
Criar um mini guia reutilizável para futuras revisões.

🧠 Engenharia de Prompts
Durante o desenvolvimento do projeto, diversos prompts foram utilizados para testar a capacidade do NotebookLM em gerar resumos, explicações e revisões de conteúdo.

✅ Prompts Utilizados
Prompt 1

Prompt 2
Gere um resumo organizado em tópicos sobre o material enviado.
Prompt 3
Quais são os principais conceitos apresentados neste conteúdo?
Prompt 4
Crie um glossário com os termos mais importantes do tema.
Prompt 5
Transforme este conteúdo em um mini guia de revisão rápida.
⚠️ Dificuldades Encontradas (Troubleshooting)
Durante os testes, algumas dificuldades foram percebidas:

Algumas respostas ficaram genéricas;
Certos resumos vieram muito extensos;
Foi necessário refinar prompts para obter respostas mais objetivas;
Em alguns momentos a IA repetiu informações;
A qualidade das respostas melhorou quando os prompts ficaram mais específicos.
✔️ Soluções Aplicadas
Utilização de prompts mais curtos e diretos;
Separação do conteúdo por tópicos;
Solicitação de respostas em formato de lista;
Revisão manual das respostas geradas.
📘 Mini Guia de Estudos
🔹 O que é o NotebookLM?
O NotebookLM é uma ferramenta de Inteligência Artificial desenvolvida pelo Google que permite analisar documentos, criar resumos, responder perguntas e organizar conteúdos de estudo utilizando IA generativa.



📝 Resumo Estruturado
Inteligência Artificial nos Estudos
A Inteligência Artificial vem sendo utilizada como apoio educacional para:

Resumos automáticos;
Explicações simplificadas;
Revisão de conteúdos;
Criação de perguntas;
Organização de materiais.
Ferramentas como o NotebookLM ajudam estudantes a economizar tempo e melhorar a compreensão dos conteúdos.



📚 Glossário
IA Generativa
Tecnologia capaz de criar textos, imagens e respostas automaticamente.

Prompt
Comando ou instrução enviada para a IA.

NotebookLM
Ferramenta do Google voltada para estudos e organização de conhecimento com IA.

Engenharia de Prompt
Processo de criação de comandos mais eficientes para obter melhores respostas da IA.

Curadoria de Conteúdo
Seleção organizada de materiais e fontes relevantes para estudo.
